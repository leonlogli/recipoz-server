# Food Recipe App with react

Recipoz is a modern web application built with apollo graphql server. 
It allows users to create, share and explore food recipes.

## Features

- Food Recipes
- Categories
- Ingredients
- Paginations
- Reviews
- Search
- Sorting & Filtering
- User management
- I18N & L10N
- Notifications

## Quick Start

```bash
# Runs the app in development mode. Open http://localhost:3000 to view it in the browser.
yarn dev
# Builds the app for production and optimizes the build for the best performance.
yarn build
# Start the server in local
yarn start
```

## Testing

```bash
yarn test
yarn test:unit
yarn test:integration
# Run the test watcher
yarn test:watch
# run test coverage
yarn test:coverage
```
